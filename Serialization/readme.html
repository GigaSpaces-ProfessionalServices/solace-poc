GigaSpaces XAP.NET HelloWorld Example

Overview

The HelloWorld example demonstrates writing a simple program using GigaSpaces XAP.NET. The program connects to a space, writes an object, and reads that object back and removes it from the space.

This example can be run with an embedded space or a remote space. An embedded space is a space which resides in your application's process. A remote space is a space which resides in another process.

Running the Example

Building is not required - the example is built automatically when run. If you wish to build manually, you can execute Build.bat.

To run the example with an embedded space, execute runEmbedded.bat.

To run the example with a remote space, first execute RunRemoteSpace.bat, wait a few seconds until you see a message saying the space was started successfully, then execute runRemote.bat.

If you prefer to use Visual Studio, you can use HelloWorld.sln. By default, running the example from Visual Studio will use an embedded space called 'EmbeddedSpace'. If you wish to run with a remote space, either type the remote space name in Project -> Properties -> Debug -> Command line arguments, or change the the code accordingly.
